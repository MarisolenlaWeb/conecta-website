Soluciones Conecta

Brindamos a nuestros clientes una plataforma de conectividad total así como servicios y aplicaciones acordes a sus necesidades y particularidades.

E-Banking

La plataforma de E-banking más avanzada en México

  • Desarrollada bajo los esquemas de seguridad y observaciones generadas por la CNBV.
  • Es una plataforma modular, escalable y segura.
  • En cada proyecto de E-banking se desarrolla un increíble UX/UI para enfocar la visión de cada institución hacia sus clientes.

M-Banking

El futuro es móvil

  • El proyecto de M-Banking de Conecta convierte a un smartphone iOS, Android o Blackberry en el canal digital más fuerte de las instituciones financieras.
  • Enrolamiento de dispositivo seguro a través de triple autenticación.
  • Notificaciones PUSH, Compartir datos de transferencia, digitalización de productos financieros al móvil.

Protect

Diferentes plataformas enfocadas en la seguridad de datos

  • ConectaCryptography: Suite de cifrado de alto nivel con dispositivos HSM. 
  • SAT: Sistema de Administración y asignación de Token.
  • MonitorHSM: Aplicación Web de administración de los procesos en tiempo real de equipos HSM.
  • TXEFT: Software de aplicación de equipos HSM tipo EFT para cifrado y validación.

Soft-Token

Virtualización de token OTP o Challenge/Response bajo las licencias de Safenet-Gemalto.

  • Este tipo de token es catalogado por la CNBV como mecanismo de autenticación categoría 3 por los altos estándares de seguridad.

SPEI

Software de conexión a la Red Financiera del Banco de México para transacciones interbancarias.

  • MotorSPEI: Sistema de comunicaciones por medio del protocolo de Banxico para las transacciones de envío y recepción de pagos.
  • SPEI Central: Aplicación de Monitoreo de pagos de envío y recepción así como el estatus de cada uno de ellos.

Switch

El proyecto Switch transacciona la compra de Tiempo Aire y de Pago de Servicios por medio de los brokers mas grandes de México para ofrecer la mayor cantidad de operaciones disponibles.
